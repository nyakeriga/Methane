# Methane (CH₄) Molecule - 3D Interactive Model

A 3D interactive visualization of the Methane molecule built with **Three.js**.

![Three.js](https://img.shields.io/badge/Three.js-v0.160.0-black?logo=three.js)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## 🔬 About

This project renders a scientifically accurate 3D model of **Methane (CH₄)** — one carbon atom bonded to four hydrogen atoms in a **tetrahedral geometry** with bond angles of approximately 109.47°.

Built as part of the **Unit 5 Assignment**.

## ✨ Features

- **1 Carbon atom** — red sphere at the center
- **4 Hydrogen atoms** — blue spheres in tetrahedral arrangement
- **4 Bond cylinders** — white connectors perpendicular to sphere surfaces
- **Green ground plane** — receives realistic shadows
- **Shadow mapping** — directional light casts soft shadows onto the plane
- **Auto-rotation** — molecule gently spins on the Y-axis
- **Mouse drag controls** — click and drag to rotate the molecule manually
- **Touch support** — works on mobile devices
- **Responsive** — adapts to any window size

## 🚀 How to Run

Simply open **`methane.html`** in any modern web browser:

```
Open methane.html in Chrome, Firefox, Edge, or Safari
```

No server, no installation, no build step required. Three.js is loaded from a CDN.
